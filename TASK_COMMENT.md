GitHub Repository: <https://github.com/suhasi-gadge/Automated-Sentiment-Analysis-Pipeline-for-News-Articles>

Completed deliverables:

- Python script: `src/news_sentiment_pipeline.py`
- Requirements file: `requirements.txt`
- Collected dataset: `data/renewable_energy_news_100.csv`
- Sentiment output dataset: `outputs/sentiment_results.csv`
- Summary report: `reports/sentiment_report.md`
- Instructions: `README.md`

The pipeline processes 100 renewable-energy news articles, performs preprocessing with tokenization, stop-word removal, and lemmatization, calculates article-level sentiment scores, stores results in CSV format, and generates a Markdown report showing sentiment distribution and top positive/negative keywords.
